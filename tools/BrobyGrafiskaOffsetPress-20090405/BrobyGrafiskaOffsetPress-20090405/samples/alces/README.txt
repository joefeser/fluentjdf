TESTING USING ALCES

1. Shift-click the SubmitQueueEntry button and select "Single Node Job.jdf"
2. Update all plates by clicking the Send File button and select "CommandResource_ExposedMedia_All.jmf"
3. Update all previews by clicking the Send File button and select "CommandResource_Preview_All.jmf"
4. The job starts printing
5. When the printing has completed the updated JDF will be returned to Alces

For more info see: http://code.google.com/p/brobygrafiska-offsetpress/wiki/TestingBrobyGrafiskaOffsetPressUsingAlces
